# SaaS Admin 使用介绍

## HTTP 响应状态

`Http Status code 200`

### 错误

#### 参数说明

|  名称  |      类型       |    说明    |
| :----: | :-------------: | :--------: |
|  code  |       int       |   业务码   |
|  msg   |     string      |  错误信息  |
| errors | array or object | 错误信息体 |

#### 错误响应

```json
{
  "code": 4022,
  "msg": "密码不能为空",
  "errors": null
}
```

### 成功响应

#### 参数说明

| 名称 |      类型       |      说明      |
| :--: | :-------------: | :------------: |
| code |       int       | 业务码: 0-正常 |
| msg  |     string      |      消息      |
| data | array or object |     消息体     |

#### 格式

```json
{
  "msg": "",
  "code": 0,
  "data": []
}
```

### 具体业务码说明

| 业务码 |        说明        |
| :----: | :----------------: |
|   0    |      请求成功      |
|  4022  | 客户端请求参数错误 |
|  4033  |  客户端请求被拒绝  |
|  5000  |     服务端错误     |
