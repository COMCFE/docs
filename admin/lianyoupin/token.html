# 通证添加

/admin/project-tokens

请求方式

POST

请求参数

名称 类型 默认 必须 说明
name string 无 否 通证名称
icon integer 无 是 通证图标
symbol string 无 是 通证符号
balance number 无 是 余额
contract string 无 否 通证合约地址，选填

SUCCESS 状态码

201

SUCCESS 返回体

{
  "message": "创建成功"
}
