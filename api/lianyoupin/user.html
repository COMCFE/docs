# 用户注册

方式

POST

路径

/api/v1/auth/register

参数

名称 类型 必须 说明
phone string 是 手机号
password string 是 密码
password_confirmation string 是 确认密码
code string 是 验证码
invite_code string 否 邀请码

响应

Status code 200

{
  "msg": "",
  "code": 0,
  "data": {
    "token": "2c2eefe290a452e51ede0253f88d94c3"
  }
}
